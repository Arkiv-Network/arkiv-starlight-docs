import { Card, CardGrid, LinkCard } from '@astrojs/starlight/components';

Build a sketch application where users draw with p5.js, save sketches to Arkiv via MetaMask, and browse their gallery. This tutorial focuses on **interacting with a browser wallet**.

[View the demo](https://arkiv-network.github.io/learn-arkiv/_demos/metamask-tutorial/) | [Source code](https://github.com/Arkiv-Network/learn-arkiv/tree/main/tutorial-source-code/metamask-tutorial)

## What You'll Build

<CardGrid>
  <Card title="MetaMask Integration" icon="star">
    Connect to the user's wallet and auto-switch to the Arkiv Kaolin testnet.
  </Card>
  <Card title="Drawing Canvas" icon="pencil">
    A p5.js-powered canvas for creating sketches.
  </Card>
  <Card title="Blockchain Storage" icon="document">
    Save drawings to Arkiv with automatic expiration.
  </Card>
  <Card title="Gallery View" icon="magnifier">
    Display all sketches owned by the connected user.
  </Card>
</CardGrid>

## What You'll Learn

- How to connect a browser wallet (MetaMask) to Arkiv
- How to switch/add the Kaolin testnet chain programmatically
- How to create entities from the browser using `custom(window.ethereum)` transport
- How to query entities by owner and display results

## Prerequisites

- Node.js installed
- MetaMask browser extension
- Basic TypeScript knowledge
- Test ETH from the [Kaolin Faucet](https://kaolin.hoodi.arkiv.network/faucet/)
